# obsidian-lifelog-templates
Obsidian で日々の記録（ライフログ）、振り返り、5年日記を効率的に行うためのテンプレート集です。

## 特徴
- **自動リンク**: ファイル名（日付）から前後の日・週・年へのリンクを自動生成します。
- **睡眠時間グラフ**: DataviewJS を使用して、睡眠時間を自動集計しグラフ化します。
- **5年日記**: 過去の同日の記録を埋め込み表示し、現在の日記と並べて振り返ることができます。
- **Todo管理**: 週のTodoをデイリーノートに埋め込むことで、タスクの漏れを防ぎます。

## 内容
以下の4つのテンプレートが含まれています。

1. **デイリーノートテンプレート** (`YYYY-MM-DD`)
    - 前後の日付へのリンク
    - 5年連用日記（過去の同日）の埋め込み
    - 今週のTODOの埋め込み
2. **ウィークリーノートテンプレート** (`GGGG-Www`)
    - 前後の週へのリンク
    - その週のデイリーノート一覧
    - 睡眠時間の推移グラフ
3. **イヤリーノートテンプレート** (`YYYY`)
    - 前年へのリンク
    - その年の睡眠時間推移グラフ
    - 筋トレ記録の埋め込み
4. **5年日記テンプレート** (`MM-DD`)
    - 過去数年分の同日の記録用ヘッダー
    - 前日へのリンク

## 使用プラグイン
- [Templater](https://github.com/SilentVoid13/Templater)
    - 日付計算や動的なコンテンツ生成に使用します。
- [Dataview](https://github.com/blacksmithgu/obsidian-dataview)
    - データの集計やグラフ描画（DataviewJS）に使用します。
- [Previsous River](https://github.com/ongaeshi/obsidian-previous-river) (Optional)
    - previous プロパティが設定されたノートを前後に移動するために使用します。


## 使い方
1. テンプレートファイルを Obsidian のテンプレートフォルダに配置します。
2. Templater の設定で、新しいファイル作成時に自動的にテンプレートを適用するように設定するか、手動でテンプレートを挿入します。
3. ファイル名は以下の形式である必要があります。
    - デイリーノート: `YYYY-MM-DD` (例: `2026-02-17`)
    - ウィークリーノート: `GGGG-Www` (例: `2026-W08`)
    - イヤリーノート: `YYYY` (例: `2026`)
    - 5年日記: `MM-DD` (例: `02-17`)
